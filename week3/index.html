You have to be careful when changing variables that share the same object in memory because you are changing the object's value. This doesn't happen with primitive values.

The this keyword refers to the object that it is within. "Namespacing is the act of wrapping a set of entities, variables, functions, objects under a single umbrella term." - flaviocopes.com. This helps prevent reusing names that might have been used somewhere else in the code.

JSON - JavaScript Object Notation. A popular data storage format. A few things to remember.

  1. Property names must be in double quotes
  2. Permitted values are double-quoted strings, numbers, true, false, null, arrays, and objects
  3. Functions are not permitted values.
The parse() method takes a string of data in JSON format and returns a JavaScript object. The stringify() method does the opposite of parse(). It takes a JavaScript object and returns it as a string of JSON data.
